# Auditor Bancario VCT — Comunidad

Material **público** del proyecto **Vanilla Center Trust [VCT]**: bot de economía en Discord para comunidades de Farming Simulator 22.

> El código del bot y el panel de administración están en el repositorio privado [App-Auditor-Bancario](https://github.com/calero1989/App-Auditor-Bancario). Aquí encontrarás documentación para jugadores y mecenas (nivel gratuito).

## ¿Qué es?

- Cuentas bancarias virtuales (NRB, cartera, caja fuerte, dinero negro)
- Nóminas por roles de Discord
- Trabajos FS22 certificados con recibos
- Juegos, submundo, mercado negro y automatizaciones 24/7
- Tiendas interactivas, alarma de seguridad, persecución y arresto (v2.9.x)
- En desarrollo: servidor de juego dedicado e integración bot ↔ partida

**Versión documentada:** **2.9.4** (Jun 2026) — parches de seguridad economía.
